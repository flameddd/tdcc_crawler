# tdcc_crawler

## pre-require
- node.js version at least v8

https://nodejs.org/en/

```bash
$ node -v
v10.15.1
```

## download, install
### clone repo

```bash
git clone https://github.com/flameddd/tdcc_crawler.git
cd tdcc_crawler
npm install
```

install need a while, library size large than 100MB

## execute
```bash
npm start
```
